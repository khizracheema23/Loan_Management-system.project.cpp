**Project Overview**

This Project is a Loan Management System developed in C++.
File Handling is used to store loan records permanently in a text file.
This system is menu-driven and easy to use for beginners.

**Features**

* Add new loan record

* View all loan records

* Update existing loan

* Delete loan record

* Data stored using file handling

**Loan Details**

* Loan ID

* Customer ID

* Loan amount

* Interest Rate

* Loan Status(Approved / Pending)

**Files Included****

 * loan_management.cpp

 * Contains the complete C++ source code.

 * loans.txt

 * Stores all loan records.This file is created automatically when the program runs.

**Files Format**

LoanID  CustID Amount InterestRate Status

101      1     50000    5.5        Approved
102      2     30000     6         Pending

**How to Run the Project (Dev-C++)**

1. Open Dev-C++

2. Create a new source file

3. Paste the C++ code

4. Save files as

5. Press F9 to compile

6. Press F10 to run
